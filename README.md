# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/satyamsahu0008/Programming-in-C-demo.git
```

### Move into the project folder

```bash
cd Programming-in-C-demo
```

### Compile a program

```bash
gcc Experiment_10.c -o program
```

### Run

**Windows**

```bash
program.exe
```

**Linux / macOS**

```bash
./program
```
